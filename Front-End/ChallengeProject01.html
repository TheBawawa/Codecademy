Selectors and Specificity

Differents Selectors Table

Definition

A selector is used to target the specific HTML element(s)to be styled by the declaration.

Selector Description Tag
Type Often referred to by their opening tag name and dont require angle brackets x (No tag, just the type)
Universal Targeting all of the elements on the page has a few specific use cases, such as resetting default browser styling *
Class It's possible to add more than one class name. You target it with the class name and period .x (Period)
ID For elements that need to be styled uniquely #x
Attribute Elements of the same type can be targeted differently by their attribute or attribute value. Surrounded by square brackets [x]
Pseudo-Class Factors such as user interaction, site navigation, and position in the document tree can all give elements a different state with pseudo-class x:y
